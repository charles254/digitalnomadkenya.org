# VizaBot KE: Digital Nomad & Work-Permit Assistant

VizaBot KE is a high-ticket "SaaS-lite" business designed to automate the Kenyan immigration process for Digital Nomads (Class N) and Investors (Class G).

## 🚀 Quick Start
To run the end-to-end workflow simulation:
```powershell
./setup.ps1
python src/orchestrator.py
```

## 🏗️ Architecture
- **`.agents/skills/immigration_pro_ke`**: Core AI logic and persona instructions.
- **`data/permits/`**: Source of truth for Immigration Act requirements (Class N, G, D).
- **`src/vision/`**: Logic for document auditing (Passport validity, Bank stamps).
- **`src/logistics/`**: High-value features (KRA PIN automation, Uptime verification).
- **`public/`**: Premium landing page with glassmorphism UI.

## 💰 Monetization Model
1. **Eligibility Pass ($10)**: Fast Vision AI audit.
2. **Application Pack ($50)**: Generated Dossier + Cover Letter + Housing.
3. **VIP Fixer ($250)**: On-ground human concierge at JKIA.

## 🛠️ Deployment (Hostinger VPS)
1. Hosted in **Germany (EU)** for GDPR compliance.
2. Secure data handling: Passport/Bank data is processed ephemerally.
3. M-Pesa Integration: Uses Daraja API for automated "Dossier" unlocks.

## 🤝 Partners
- **Airbnb Owners**: Vetted properties with Solar + Starlink.
- **Human Fixers**: Ground partners for terminal-side assistance.
